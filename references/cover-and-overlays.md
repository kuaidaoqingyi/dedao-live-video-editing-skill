# Cover And Overlays

## Cover Rules

- Use a bright person-first frame when available.
- Prefer open eyes, clear face, strong emotion, or a frame where the object/book is visible.
- Main title defaults to two centered lines.
- Avoid small explanatory text unless the user asks.
- Use large high-contrast text; if the title merges with the background, add a very transparent dark mask behind the text.
- Save every version; never overwrite a cover the user has seen.

Naming:

```text
frames/cover-options/v01-eye-open/option-01.jpg
frames/cover-options/v02-emotion/option-01.jpg
versions/covers/cover-v03-title-style.jpg
```

## In-Video Assets

Use picture-in-picture for:

- guest photos
- book/product images
- screenshots
- supporting B-roll
- short external video assets

Placement:

- Keep away from speaker eyes and mouth.
- Keep away from subtitle zone.
- If the host is crying, pausing, or showing strong emotion, reduce overlays.
- If inserted video has useful original audio, mix intentionally and document volume.

## Quote Cards And 花字

- Use full-screen quote cards only for major written lines or key concepts.
- For emotional sections, prefer top or side quote cards that do not block the face.
- Do not use full-screen text to hard-sell a product.
- Sync 花字 with speech; do not show the punchline too early.
- Highlight nouns, names, concrete concepts, and strong verbs. Avoid highlighting empty function words.

## HyperFrames Notes

Use HyperFrames when the edit needs:

- timed HTML/CSS captions
- PIP images or videos
- quote cards
- cover-first-frame treatment
- final end card

Prefer one final encode. Avoid repeatedly burning and re-encoding the same video through several tools.
